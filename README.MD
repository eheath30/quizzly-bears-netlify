# QuizzlyBears
## LAP 3 Project Assignment
This repo contains the client for a fullstack project built with reactjs, redux, socket.io, nodejs, expressjs and pgSQL.


## Installation & Usage
### Installation
- cd into quizly-bears-client
- run: `npm i`

### Usage
- cd into quizly-bears-client
- run `npm run dev`
